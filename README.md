# 🌐 Portfolio Website

## 📌 Project Description

The **Portfolio Website** is a responsive and modern personal portfolio
built using HTML, CSS, and JavaScript. It is designed to showcase
personal information, skills, projects, and contact details in a clean
and professional layout. The website works seamlessly across desktop,
tablet, and mobile devices.

------------------------------------------------------------------------

## 🚀 Technologies Used

-   HTML5\
-   CSS3\
-   JavaScript

------------------------------------------------------------------------

## ✨ Features

-   ✅ Responsive design (Mobile-friendly)
-   ✅ Navigation bar with smooth scrolling
-   ✅ About section to showcase profile details
-   ✅ Contact form for user inquiries
-   ✅ Clean and modern UI layout
